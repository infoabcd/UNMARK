# 提示詞選擇

UNMARK 會讀取 `Prompts/` 內所有 `.md` 檔案。檔名是不包括 `.md` 的選項名稱，檔案內容會完整送到模型。

內置選項：

| 選項 | 適用情況 |
|------|----------|
| `萬能提示詞` | 預設。混合圖片或不確定水印類型時使用。 |
| `樓盤Logo` | 地產相片、平面圖、代理 Logo、網址及電話水印。 |
| `二維碼` | 疊加式二維碼及其底色、邊框或附帶文字。 |
| `亂塗亂畫` | 後加筆畫、圈選、箭嘴、塗鴉及標記線。 |
| `線稿粗黑筆` | 鉛筆畫、漫畫或黑白線稿上，與幼線有明顯粗幼差異的實心黑筆覆蓋。 |
| `密集文字水印` | 重複文字、圖庫名稱、半透明作者名及角落去水印標籤。 |

## 自己新增

1. 在 `Prompts/` 建立 UTF-8 編碼的 `.md` 檔案，例如 `商品Logo.md`。
2. 檔案內只寫要送給模型的提示詞，不要加入標題、Front Matter 或說明文字；檔案不可留空或超過 20,000 個字元。
3. 重新載入 LocalWeb 的設定頁或新增任務頁，選項會顯示為 `商品Logo`。

命令列可用檔名選擇：

```bash
python3 run_unmark.py --list-prompts
python3 run_unmark.py run --prompt-preset 商品Logo
```

長期預設值可放入 `.env`：

```env
UNMARK_PROMPT_PRESET=萬能提示詞
```

本地 API 的 JSON 或 multipart 表單同樣接受 `prompt_preset`。名稱必須對應 `Prompts/` 內現存的 `.md` 檔案。

## 失敗對照

[失敗案例](Examples/failures/README.md) 保留了萬能提示詞漏除粗黑筆，以及樓盤相片誤刪真實標誌的代表性對照。提示詞可以降低誤判，但不能取代人工抽查。
